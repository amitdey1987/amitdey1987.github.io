I am new to the Mac environment and still learning to use it effectively. This post focucess on Visual Studio Code and it's shortcuts in Mac.

[Official Visual Studio Code Shortcuts](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf)

# General
|Action|Shortcut|
|-|-|
|Open|⌘ + o|
|New File|⌘ + n|
|New File|⌘ + w|

# Editing
|Action|Shortcut|
|-|-|
|Cut|⌘ + x|
|Copy|⌘ + c|
|Paste|⌘ + v|
|Delete Line|⌘ + Shift + k|
|Indent|Tab|
|Begin/End Line|Home/End|
|Scroll Up/Down|Page Up/Page Down|
|Fold/Unfold|Option + ⌘ [/[|
|Toggle Comment|⌘ + /|

# Search
|Action|Shortcut|
|-|-|
|Find|⌘ + f|
|Replace|Option + ⌘ + f|
|In Search, Next/Previous|⌘ + g/Shift + ⌘ + g|
|In Search, Highlight All|Option + Enter|
|Find Reference|Shift + F12|
|Rename Symbol|F2|

# Navigation
|Action|Shortcut|
|-|-|
|Go to line|Control + G|
|Go to symbol|Shift + ⌘ + o|
|Go to next/previous warning|F8/Shift + F8|
|Go Back/Forward|Control + -/Shift + Control + -|
